Data

Raw data

The data were obtained from five different, mostly repeated experiments, and were organized in a single xlsx file. Data from each experiment are in a separate sheet, which are loaded separately using the read_excel function of the readxl file.

I found much more convenient to use this binary file instead of storing in a text file like csv. For some variabeles, I had to deal with some characters which were not loading correctly when usingcsvfiles.

Most of the data are already organized in the long format, but some are in the wide, and so will need some transformation. These are done within the respective section of the data analysis page.
